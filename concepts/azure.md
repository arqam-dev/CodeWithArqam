<expand title="Core Azure Services">
## Core Azure Services

### Compute Services

- **Virtual Machines**: Infrastructure-as-a-Service (IaaS)
  - Flexible virtual server options
  - Support for Windows and Linux

- **App Service**: Platform-as-a-Service for web apps
  - Deploy and scale web applications
  - Supports multiple programming languages

- **Azure Functions**: Serverless functions (similar to AWS Lambda)
  - Event-driven compute service
  - Pay only for execution time

- **Azure Container Instances**: Container hosting
  - Run containers without managing servers

- **Azure Kubernetes Service (AKS)**: Managed Kubernetes
  - Deploy, scale, and manage containerized applications

### Storage Services

- **Blob Storage**: Object storage (similar to AWS S3)
  - Store unstructured data
  - Scalable and cost-effective

- **Azure Files**: Managed file shares
  - SMB and NFS file shares
  - Accessible from cloud or on-premises

- **Disk Storage**: Block storage (similar to AWS EBS)
  - Persistent storage for virtual machines

- **Data Lake Storage**: Big data storage
  - Optimized for analytics workloads

### Database Services

- **SQL Database**: Managed SQL database
  - Fully managed relational database
  - High availability and automatic backups

- **Cosmos DB**: Globally distributed NoSQL database
  - Multi-model database service
  - Low latency and high availability

- **Azure Database for PostgreSQL/MySQL**: Managed relational databases
  - Fully managed open-source databases

- **Redis Cache**: In-memory caching
  - High-performance caching solution

### Networking Services

- **Virtual Network (VNet)**: Isolated network environment
  - Create private networks in Azure
  - Connect to on-premises networks

- **Load Balancer**: Traffic distribution
  - Distribute incoming traffic across multiple resources

- **Azure CDN**: Content delivery network
  - Deliver content with low latency

- **Azure DNS**: DNS hosting
  - Host your DNS domains in Azure

### Identity & Security Services

- **Azure Active Directory (Azure AD)**: Identity and access management
  - Single sign-on and multi-factor authentication
  - Integration with Microsoft services

- **Key Vault**: Secrets management
  - Securely store and manage secrets, keys, and certificates

- **Security Center**: Security monitoring
  - Unified security management and threat protection

- **Azure Sentinel**: Security information and event management (SIEM)
  - Cloud-native SIEM solution

### Integration Services

- **Service Bus**: Messaging service
  - Reliable messaging for applications

- **Event Grid**: Event routing service
  - Event-driven architecture support

- **API Management**: API gateway
  - Publish, secure, and manage APIs

### Analytics Services

- **Azure Synapse Analytics**: Data warehouse
  - Big data analytics and data warehousing

- **HDInsight**: Big data analytics
  - Managed Hadoop, Spark, and other big data services

- **Azure Stream Analytics**: Real-time stream processing
  - Process streaming data in real-time

</expand>

## Secondary Concepts

<expand title="Azure vs AWS">
## Azure vs AWS

- **Enterprise Integration**: Azure better for Microsoft environments
  - Seamless integration with existing Microsoft infrastructure
  - Native Active Directory support

- **Hybrid Cloud**: Azure stronger hybrid capabilities
  - Better integration between on-premises and cloud
  - Azure Arc for managing resources across environments

- **Pricing**: Azure often competitive, especially for Windows workloads
  - Cost-effective for Microsoft-based organizations

- **Market Position**: AWS leads, Azure second in market share
  - Growing rapidly in enterprise market

- **Open Source**: AWS traditionally stronger, Azure improving
  - Azure now has strong open-source support

- **Compliance**: Azure strong in enterprise compliance certifications
  - Extensive compliance offerings for regulated industries

</expand>

<expand title="Cloud Provider Comparison">
## Cloud Provider Comparison

- **AWS**: Largest market share, comprehensive services, enterprise-ready
- **GCP**: Strong ML/AI, data analytics, Kubernetes-native

- **Choosing**: Depends on existing infrastructure, tech stack, and specific needs
