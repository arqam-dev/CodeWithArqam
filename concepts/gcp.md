<expand title="General Points">
## General Points

- Google Cloud Platform (GCP): Suite of cloud computing services provided by Google
- Provides infrastructure, platform, and serverless services
- Pay-as-you-go pricing model
- Strong focus on data analytics and machine learning
- Global network infrastructure
- Kubernetes-native (originator of Kubernetes)
- Open source friendly

</expand>

<expand title="Core GCP Services">
## Core GCP Services

### Compute Services

- **Compute Engine**: Virtual machines (similar to AWS EC2)
  - Flexible, high-performance virtual machines
  - Custom machine types available

- **App Engine**: Platform-as-a-Service (PaaS)
  - Fully managed application platform
  - Automatic scaling and load balancing

- **Cloud Functions**: Serverless functions (similar to AWS Lambda)
  - Event-driven serverless compute
  - Pay only for execution time

- **Cloud Run**: Serverless container platform
  - Run containerized applications without managing infrastructure

### Storage Services

- **Cloud Storage**: Object storage (similar to AWS S3)
  - Scalable object storage
  - Multiple storage classes available

- **Persistent Disk**: Block storage (similar to AWS EBS)
  - High-performance block storage for VMs

- **Cloud Filestore**: Managed file storage
  - Fully managed NFS file servers

### Database Services

- **Cloud SQL**: Managed relational databases
  - MySQL, PostgreSQL, SQL Server
  - Automatic backups and replication

- **Firestore**: NoSQL document database
  - Serverless document database
  - Real-time synchronization

- **Bigtable**: NoSQL wide-column database
  - High-performance NoSQL database
  - Ideal for large-scale applications

- **Cloud Spanner**: Globally distributed relational database
  - Horizontally scalable relational database
  - Strong consistency across regions

### Networking Services

- **Cloud Load Balancing**: Load distribution
  - Distribute traffic across multiple instances
  - Global and regional load balancing

- **Cloud CDN**: Content delivery network
  - Low-latency content delivery
  - Global edge network

- **VPC**: Virtual Private Cloud
  - Isolated network environment
  - Custom network topology

### Big Data & Analytics Services

- **BigQuery**: Serverless data warehouse
  - Analyze petabytes of data
  - SQL-like queries on large datasets

- **Dataflow**: Stream and batch data processing
  - Unified stream and batch processing
  - Apache Beam-based

- **Pub/Sub**: Messaging service
  - Reliable messaging for event-driven systems
  - Global message delivery

### AI/ML Services

- **Vertex AI**: Machine learning platform
  - End-to-end ML platform
  - AutoML and custom model training

- **AutoML**: Automated machine learning
  - Train custom models without ML expertise
  - Multiple data types supported

- **Cloud Vision API**: Image analysis
  - Detect objects, faces, and text in images
  - OCR capabilities

- **Cloud Speech-to-Text**: Speech recognition
  - Convert audio to text
  - Multiple languages supported

### Identity & Security Services

- **Identity and Access Management (IAM)**: Access control
  - Fine-grained access control
  - Role-based access management

- **Cloud Identity**: Identity management
  - User and device management
  - Single sign-on capabilities

- **Cloud Security Command Center**: Security monitoring
  - Security and risk management platform
  - Threat detection and prevention

</expand>

## Secondary Concepts

<expand title="GCP vs AWS">
## GCP vs AWS

- **Pricing**: GCP often more competitive, especially for compute
  - Sustained use discounts
  - Per-minute billing

- **Machine Learning**: GCP has stronger ML/AI services
  - Vertex AI platform
  - Strong AutoML capabilities

- **Kubernetes**: GCP originated Kubernetes, strong native support
  - Google Kubernetes Engine (GKE)
  - Best-in-class Kubernetes support

- **Global Network**: GCP's private fiber network
  - Low-latency global network
  - Extensive edge locations

- **Market Share**: AWS leads, but GCP growing rapidly
  - Strong growth in enterprise market

- **Database**: GCP has unique offerings like Spanner (globally distributed SQL)
  - Cloud Spanner: Unique globally distributed relational database
  - Strong NoSQL options (Firestore, Bigtable)

</expand>

<expand title="Cloud Provider Comparison">
## Cloud Provider Comparison

- **AWS**: Largest market share, comprehensive services, enterprise-ready
- **GCP**: Strong ML/AI, data analytics, Kubernetes-native
- **Azure**: Best Microsoft integration, hybrid cloud, enterprise focus
- **Choosing**: Depends on existing infrastructure, tech stack, and specific needs
